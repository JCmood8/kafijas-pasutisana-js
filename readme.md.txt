# Kafijas Pasūtīšanas Mājaslapa


## Projekta apraksts
Šī ir interaktīva mājaslapa kafijas pasūtīšanai. Lietotājs var izvēlēties kafijas veidu un izmēru, un sistēma aprēķina cenu.


## Funkcionalitāte
- Lietotājs izvēlas kafiju un izmēru.
- JavaScript nolasa ievadi.
- Tiek veikts cenas aprēķins.
- Ja cena pārsniedz 5€, tiek piemērota 10% atlaide.
- Rezultāts tiek parādīts dinamiski bez lapas pārlādes.


## Lietošanas instrukcija
1. Izvēlies kafijas veidu.
2. Izvēlies izmēru.
3. Nospied pogu "Aprēķināt cenu".
4. Rezultāts tiks parādīts zem pogas.
